BGA game record converter

Instructions

  1. Go to a game review page like this one.
  2. Locate the text record of the game, like the one pictured (you'll need to scroll down a little).
  3. Copy the game record .
  4. Paste the entire record into the box below. The timestamps will be included, which is fine.
  5. Click the "Convert" button below.
  6. Scroll down to see the record converted to several styles.
    • The condensed BGA format has the timestamps and turn restarts removed.
    • The SDG format is fairly compact but easy to follow while reproducing the game with physical pieces.
    • The HWL format may be imported into the HWL sandbox for smooth visual game review.

Known limitations

  • Information about the game result (winner/draw) is ignored.
  • You can't skip any part of a line (the system will break if you miss the final period, for example).
  • The "creation" phase of the game (placing homeworlds) must be completed or the converter will crash.
  • Some player names could break the system, like if your name is "restarts their turn"
